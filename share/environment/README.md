# Reference Environment

This directory stores reproducibility snapshots for the local `IR3` conda environment used to build and test `G4CAD`.

- `IR3.environment.yml` is a human-readable export that is usually the best starting point for recreating a similar environment.
- `IR3.explicit.txt` is a stricter package manifest generated by `conda list --explicit`.

Typical usage:

```bash
conda env create -f IR3.environment.yml
conda create --name IR3-repro --file IR3.explicit.txt
```

The explicit file keeps the original package URLs, including the conda mirrors configured on the export machine.
